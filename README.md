# DINODistillery
A library for distilling DINOv3 models for use in mobile and embedded systems


## Models

| Model  | Distillation             | Params(M) |
| nano   | vit small+ -> RepVit-M1  | |
| small  | vit base -> RepVit-M1.1  | |
| medium | vit base -> RepVit-M1.5  | |
| large  | vit large -> RepVit-M2.3 | |
